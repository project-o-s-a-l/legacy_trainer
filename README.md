# Legacy Trainer

## Основная идея проекта
Веб-приложение, где пользователю дают кусок плохо написанного, запутанного кода (Legacy кода), с некоторыми багами на каком либо языке (Изначально планируеться поддержка pythonб С#). Пользователю нужно:
- Найти и исправить баги
- Добавить нову фичу в эту запутанную структуру
- Написать тесты для этого кода

## Фишка:
- Оценка качества кода от ИИ 

## Стек используемых технологий:
- Python (Fast API, SQLAlchemy, psycopg2, pytest, g4f) - Оценка кода(работа с ИИ), работа с БД.
- C# (ASP.NET Web API,unit test) - Принимает request от фронтенда, запускает код, делает тесты и отправляет response питону на оценку
- Docker
- Фронтенд: React.js, HTML, CSS
- База данных - PostgreSQL
